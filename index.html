SIQ025 - Software de modelización de sistemas industriales¶

Bienvenid@s!

Esta es la parte 2.2 de la asignatura SIQ025 - Software de modelización de sistemas industriales del máster en Matemática computacional de la UJI.

En esta parte de la asignatura aprenderemos, desde un punto de vista más práctico, cómo podemos gestionar desde la nube todo el software que vayamos desarrollando durante nuestra carrera profesional. También veremos cómo se pueden automatizar tareas que se tienen que ejecutar cada vez que se modifique el código (por ejemplo, la actualización de la página web del paquete).

Lo que se pretende conseguir al final de esta parte es que seais capaces de implementar un paquete de software (en este caso del lenguaje de programación de R) y gestionarlo desde la nube.

En primer lugar, en la lección 1 veremos qué es un paquete de R, analizaremos sus principales partes y crearemos un paquete de R muy simple usando RStudio.

A continuación, en la lección 2 aprenderemos a gestionar nuestro paquete desde la nube usando el sistema de control de versiones git y el entorno web de Github.

Después, en la lección 3 documentaremos todas las funciones de nuestro paquete para facilitar a cualquier usuario su uso.

Finalmente, en la lección 4 aprenderemos a automatizar tareas desde Github usando una CI/CD.
